Bienvenido a la documentación de **essential-materials**. Esta guía te ayudará a configurar e integrar la librería en diferentes entornos de desarrollo de React.

### Instalación

Para comenzar, instala la librería con **npm** o **yarn**:

```bash
npm install essential-materials
```

```bash
yarn add essential-materials
```

Si usas **pnpm**, puedes instalarlo con:

```bash
pnpm add essential-materials
```

Luego, asegúrate de que tu proyecto soporte **CSS Modules** o **CSS estándar**, ya que la librería usa estilos CSS. Puedes importar los estilos globalmente en tu proyecto:

```tsx
import "essential-materials/dist/index.css";
```
